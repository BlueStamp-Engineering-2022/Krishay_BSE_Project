This file contains a class called VideoCamera that uses OpenCV to define a variety of methods for object detection.

One important method used here is the get_frame() method, as it returns a jpeg image in the form of bytes. That jpeg image contains the camera's current frame.

Another important method is the get_object() method, which simply just returns the jpeg image in bytes as well as a boolean that shows whether an object was found or not.
