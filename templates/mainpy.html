Whenever the user wants to run the camera, this file should be run within a Python 2.7 environment.

One important method in this file is the check_for_objects() method, which does the same thing as its name. Once something is detected, it checks whether an email was sent previuosly within a given time interval. If an object was detected and no email was sent recently, it sends the email with the information using the sendEmail() method.
